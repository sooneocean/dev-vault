# CLI Quick Reference 缺少 v0.7.0 新命令

## Problem

Journal 2026-03-29 記錄升級 obsidian-agent v0.6.1 → v0.7.0，新增 6 個命令（stale, cluster, digest, thread, suggest, context），但 CLAUDE.md 的 CLI Quick Reference 段落仍只列出原始命令。Agent 在使用時不知道這些新能力可用。

## Suggested Change

在 CLAUDE.md 的 CLI Quick Reference ```` ```bash ```` 區塊中新增：

```
obsidian-agent stale                     # find stale notes
obsidian-agent cluster                   # topic clustering
obsidian-agent digest                    # daily digest
obsidian-agent thread "topic"            # follow a topic thread
obsidian-agent suggest                   # smart action suggestions
obsidian-agent context "note"            # get note context
```

## Expected Effect

Agent 能知道並主動使用 v0.7.0 的新能力（如 suggest 產生建議、stale 找過期筆記、cluster 做主題分群），提升 vault 管理效率。

## Application Record

- Applied: 2026-03-29
- Target: `CLAUDE.md` CLI Quick Reference section
- Change: Added 6 v0.7.0 commands (stale, cluster, digest, thread, suggest, context)
