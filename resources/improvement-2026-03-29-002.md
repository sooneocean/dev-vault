# CLAUDE.md 缺少 docs/ 目錄說明

## Problem

安裝 Compound Engineering Plugin 後，`/ce:plan` 產生的計畫存放在 `docs/plans/`，`/ce:compound` 的 learnings 存放在 `docs/solutions/`。但 CLAUDE.md 的 Directory Structure 表格只列出 PARA 目錄（areas/, projects/, resources/, journal/, ideas/），沒有提及 `docs/`。Agent 可能不知道去哪裡找計畫文件或 learnings。

## Suggested Change

在 CLAUDE.md 的 Directory Structure 表格後加一個說明：

```markdown
| `docs/`     | docs     | Plans, solutions & learnings  |
```

或在表格後補充：

```markdown
Additional directories (from plugins):
- `docs/plans/` — Implementation plans (`/ce:plan`)
- `docs/solutions/` — Solved problem learnings (`/ce:compound`)
```

## Expected Effect

Agent 知道 `docs/` 目錄的用途，在需要查找計畫或過去的解決方案時能正確導航。
