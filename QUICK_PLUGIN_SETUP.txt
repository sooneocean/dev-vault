⚠️  TROUBLESHOOTING

Q: Don't see green notification?
A: 1. Refresh page (F5)
   2. Clear browser cache
   3. Log out and log back in
   4. Check Plugins page - plugin should show "Active"

Q: Widgets don't show?
A: 1. Clear all caches
   2. Hard refresh page (Ctrl+F5)
   3. Check Appearance > Widgets
   4. Verify theme has footer widget area

Q: Breadcrumbs don't show?
A: 1. Check Yoast SEO installed & active
   2. Check Breadcrumbs enabled in Yoast settings
   3. Clear cache and hard refresh
   4. Wait 1-3 days for Google re-crawl

Q: How to uninstall?
A: 1. Plugins > YOLO LAB Auto Deploy
   2. Click: Deactivate
   3. Click: Delete
   4. All settings preserved
