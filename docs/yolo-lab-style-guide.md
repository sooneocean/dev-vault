# YOLO LAB 文案風格指南 - 人類化改寫標準

**版本**: 1.0
**建立日期**: 2026-04-01
**適用對象**: SEO 文案、標題、Meta 描述
**參考基準**: Batch 1-2 人類化重寫分析

---

## 一、核心原則

### 1. 對話性優先
- 寫得像跟朋友聊天，不像企業官方聲明
- 使用第二人稱「你」而非「您」（除非特定場景）
- 使用「我們」建立共同體感

**例子**:
- ✓ 「到底哪些值得看？我們找出 2026 必追韓劇」
- ✗ 「幫助您快速找到最適合的必看好劇」

### 2. 台灣在地化
- 優先使用台灣詞彙和俚語
- 避免過度簡體中文或中文通用詞

**台灣詞彙清單**:
- 「誰贏」vs「排名」
- 「最威」vs「最優」
- 「怎麼玩」vs「如何使用」
- 「夠用」vs「適合」
- 「划算」vs「性價比」
- 「坑」vs「陷阱」
- 「爆料」vs「揭露」
- 「扒開」vs「深層分析」
- 「拆解」vs「分析」
- 「大作」vs「作品」
- 「廠牌」vs「品牌」
- 「上班族」vs「職場人士」

### 3. 數字錨點
- 盡量使用具體數字而非模糊描述
- 開場時加入量化數據製造視覺焦點

**例子**:
- ✓ 「月入 10 萬真的假的？」
- ✗ 「帶貨收入情況分析」

### 4. 去模板化
- 避免重複使用相同詞彙
- 每篇文案要有獨特的開場和語調

**禁用詞** (出現超過 2 次):
- ~~幫助您~~ / ~~助您~~ / ~~幫您~~
- ~~完整~~ / ~~詳細~~ / ~~全面~~
- ~~評測~~ (改用「測」「拆解」「對比」)
- ~~分析~~ (改用「拆解」「掘」「扒開」)
- ~~推薦~~ (改用「最推」「最威」「必看」)
- ~~指南~~ / ~~教學~~ (改用「入門」「怎麼選」「101」)

## 二、標題改寫公式

### 類型 1: 對比/PK 類 (vs, 誰贏, 怎麼選)

**公式**: `[工具/品牌1] vs [工具/品牌2] vs [工具/品牌3]：[年份] [商品] [懷疑詞]？`

**懷疑詞選項**:
- 誰最強？
- 誰贏？
- 怎麼選？
- 誰最全能？
- 哪個最夠用？
- 為什麼選它？

**例子**:
- Netflix 韓劇: 「2026 年最值得看的韓劇？Netflix 新劇排行」
- AI 工具: 「Midjourney vs DALL-E vs Stable Diffusion：2026 AI 畫圖工具誰最強？」
- 遊戲本: 「遊戲本 2026 怎麼選？ROG vs MSI vs ASUS 效能對決」

### 類型 2: 排名/評測類 (排行, 大評測, 對決)

**公式**: `[商品] [年份]：[懷疑詞] [具體需求]？`

**懷疑詞選項**:
- 哪家最...？
- 怎麼選？
- 大評測：[需求]？
- 排行：[對比需求]？

**例子**:
- 便當: 「便利商店便當熱量排行 2026：哪家最低卡、最健康？」
- 咖啡廳: 「台灣連鎖咖啡廳排名：環境、服務、咖啡，誰最全能？」
- 瘦身: 「2026 瘦身方法大評測：哪種減脂最有效？」

### 類型 3: 數字/金額類 (賺多少, 月入, 費用)

**公式**: `[動作詞] [商品] [數字/金額]？[年份] [補充詞]`

**動作詞選項**:
- 賺多少
- 月入
- 花多少
- 成本
- 價格

**例子**:
- 直播帶貨: 「抖音直播帶貨賺多少？2026 達人收入大拆解」

### 類型 4: 新手/教學類 (入門, 101, 從 0 開始)

**公式**: `[商品] [新手詞] [年份]：[具體問題]？`

**新手詞選項**:
- 新手入門
- 從 0 開始
- 101
- 傻瓜指南

**例子**:
- 股票: 「股票投資新手入門 2026：從 0 開始學什麼？」

## 三、Meta 描述改寫公式

### 開場必須 5 選 1

1. **痛點提問**: 「[主題] [常見困境]？」
   - 「Netflix 新劇眾多，到底哪些值得看？」
   - 「上班族吃便當怕發胖？」
   - 「股票市場新手總是被嚇到？」

2. **數字焦點**: 「[商品] [具體數字/量詞] [質疑詞]？」
   - 「抖音帶貨主播月入 10 萬真的假的？」
   - 「減肥方法千百種，到底哪種最有效？」

3. **選擇困境**: 「[場景] [品牌眾多]？」
   - 「想用 AI 畫圖但不知道選哪個？」
   - 「找工作地點頻常糾結？」
   - 「想買遊戲本但品牌一堆？」

4. **場景代入**: 「[用戶角色] [常見狀況]？」
   - 「想學新技能卻不知道上哪個平台？」

5. **故事開場**: 「[主題] [對比/狀況]」
   - 「台灣夜市美食戰爭火爆」

### 主體結構 (3 部分)

**第一部分**: 轉移焦點 + 我們主動

```
我們 [動詞] + [具體方法]
- 「我們找出 2026 必追韓劇」
- 「我們實地體驗、價格比較、評分分析」
- 「我們測了各家連鎖便當的熱量、營養」
- 「我們用科學數據比較」
- 「我們扒開數字、分析變現模式」
```

**第二部分**: 關鍵詞 + 具體指標

```
[指標1]、[指標2]、[指標3]
- 「功能、價格、效果」
- 「熱量、營養」
- 「環境、服務、咖啡品質」
- 「課程質量、師資、價格」
```

**第三部分**: 結果導向 + 情感化承諾

```
[讓/幫/教/告訴] + [使用者] + [具體結果]
- 「讓你不再踩雷」
- 「幫你找到必吃名單」
- 「讓你找到最夠用的那個」
- 「讓你安心踏入股市」
- 「教你健康瘦身不復胖」
```

### Meta 範本

```
[開場提問] → [我們的行動] + [具體指標] → [承諾結果]

例：
「Netflix 新劇眾多，到底哪些值得看？我們找出 2026 必追韓劇，從劇情、演員到評分全面拆解，讓你不再踩雷。」

分解：
- 開場：Netflix 新劇眾多，到底哪些值得看？
- 行動：我們找出 2026 必追韓劇
- 指標：劇情、演員、評分
- 承諾：全面拆解，讓你不再踩雷
```

## 四、禁用詞大全與替代方案

### 企業評估詞彙

| 禁用詞 | 替代方案 | 使用場景 |
|------|--------|--------|
| 完整 | 全面、全部、完全、徹底 | 「全面拆解」「徹底比較」 |
| 詳細 | (移除，用具體詞) | 改用「從 X 到 Y」或具體指標 |
| 深度分析 | 拆解、掘、扒開、大解析 | 「全面拆解」「大解析」 |
| 評測 | 測、測試、對比、試過 | 「我們測了」「實測結果」 |
| 評比 | 對比、比一比、對決、打 | 「全面對比」「三強對決」 |
| 指南 | 入門、101、怎麼選、教你 | 「入門 101」「怎麼選指南」 |
| 教學 | 入門、從 0 開始、怎麼做 | 「新手入門」「從 0 開始」 |
| 精選 | 挑選、找出、整理、推薦 | 「我們挑選」「精心整理」 |
| 推薦 | 最推、首選、必看、值得 | 「上班族最推」「健身愛好者必看」 |
| 發現 | 找出、挖掘、掌握、學到 | 「找出最低卡選項」 |
| 揭秘 | 扒開、拆解、分析、掘、看 | 「扒開月收數字」「拆解變現模式」 |

### 對沖與模糊詞

| 禁用詞 | 替代方案 | 例子 |
|------|--------|-----|
| 幫助您 / 助您 / 幫您 | (改用「我們」主動句) | 「讓你不再踩雷」 |
| 可以 | (直接陳述) | 「幫你找到最低卡選項」 |
| 一覽 | (移除，改用列舉) | 「熱量、營養、CP 值」 |
| 快速 | (具體詞) | 改用「立刻」「馬上」 |
| 最適合 | 最推、最威、最夠用、最順 | 「找到最順的那台」 |
| 首選 | 最推、Top 1、必選 | 「上班族最推的選擇」 |

### 承諾詞謹慎使用

| 避免用語 | 改善方案 |
|---------|--------|
| 「累積被動收入」 | 「踏入股市」「開始投資」 |
| 「安全進入」「無風險」 | 「安心踏入」「理解風險」「懂得控制」 |
| 「保證有效」「絕對」 | 「最有效」「科學驗證」「對比結果」 |
| 「100% 滿足」 | 「找到最適合你的」「都有優缺點」 |

## 五、動詞選擇指南

### 強動詞 (增加活力)

```
拆解、掘、扒開、打、對決、對戰、比拼、pk、
測試、實驗、驗證、對比、挑選、整理、統計、
分析、看、聊、說、教、告訴、分享
```

### 弱動詞 (避免)

```
分析、評比、評測、推薦、揭秘、介紹、闡述、
呈現、展示、闡明、講述
```

## 六、數字與量化的使用

### 標題中的數字

```
✓ 具體量詞：「月入 10 萬」「100 種減肥法」「3 大廠牌」
✓ 年份：「2026 年」「2025 最新」
✓ 排名：「Top 5」「排行」「第一名」
✗ 模糊：「眾多」「多種」「很多」
```

### Meta 中的數字

```
✓ 開場數字製造視覺焦點：「月入 10 萬」「減肥方法千百種」
✓ 指標具體化：「最低卡」「最划算」「3A 大作」
✗ 籠統描述：「眾多課程」「多個選擇」
```

## 七、避免的寫法

### 1. 全 Title Case
- ✗ 「Netflix 2025 必看韓劇推薦」
- ✓ 「2026 年最值得看的韓劇？」

### 2. Em dash (—) 濫用
- ✗ 「直播帶貨時代 — 抖音達人收入分析」
- ✓ 「抖音直播帶貨賺多少？」

### 3. 過度冒號
- ✗ 「標題：副標題 - 詳細說明」
- ✓ 「標題：簡潔副標題」

### 4. 聊天機器人語言
- ✗ 「I hope this helps!」(中文版「希望幫到你」)
- ✓ 自然對話結尾

### 5. 過度列舉
- ✗ 「詳細分析 X、Y、Z 三個維度，深度拆解，全面評比」
- ✓ 「拆解環境、服務、咖啡品質」

## 八、類別特定指南

### 科技/工具類
- 使用「最強」「最夠用」「效能對決」
- 加入具體功能對比
- 開場：「想要 X 但不知道選哪個？」

### 美食/餐飲類
- 使用「最威」「誰贏」「激戰」
- 加入「實地體驗」製造信度
- 開場：「到底哪家...？」

### 金融/投資類
- 使用「被嚇到」等情感詞，降低門檻心理
- 謹慎承諾（不保證收益）
- 開場：「新手總是被嚇到？」

### 健身/減肥類
- 針對痛點（發胖、復胖恐懼）
- 開場：「怕...？」
- 避免激進承諾

### 教育/學習類
- 使用「從 0 開始」「入門」
- 強調師資與質量
- 開場：「不知道選哪個平台？」

## 九、檢查清單

在發布前，逐項檢查：

- [ ] 標題中有懷疑詞或問句嗎？(「誰」「怎麼」「真的嗎」)
- [ ] Meta 開場是痛點提問嗎？
- [ ] 是否使用了「我們」主動句式？
- [ ] 是否移除了「幫助您」「完整」「詳細」？
- [ ] 是否有台灣詞彙？(最威、怎麼玩、夠用等)
- [ ] 動詞是強動詞嗎？(拆解、對決、測試)
- [ ] 是否有具體數字或量化指標？
- [ ] 字數是否簡潔？(標題 < 20 字，Meta < 160 字)
- [ ] 語氣是對話化嗎？(像朋友聊天)
- [ ] 是否避免了企業官方語氣？
