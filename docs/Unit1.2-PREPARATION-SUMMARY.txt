COMMON ISSUES & QUICK FIXES
================================================================================

If Yoast Schema not applying after config:
  → Clear Yoast cache (Tools → File Cleanup)
  → Re-save any article
  → Wait 10 minutes
  → Verify in Yoast Panel

If Google Rich Results Test returns no results:
  → Refresh page (Ctrl+F5)
  → Wait 10-15 minutes (Google crawler delay)
  → Try different URL format (www vs non-www)

If image URL fails validation:
  → Edit article → Update featured image
  → Ensure image URL starts with https://
  → Save article
  → Re-test after 10 minutes
