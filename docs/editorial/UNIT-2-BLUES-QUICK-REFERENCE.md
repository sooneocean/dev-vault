**必備** ✓
- 搖滾或民族音樂背景

- 能寫出「力量感」，不只悲傷

**加分** ✓
- 對伍佰有認識
- 對台灣 1990s 音樂文化熟悉

**禁區** ✗

- 學術風格（樂理術語）
- 企業文案風格（官方語氣）

## 關鍵會議 — 規劃會議檢查表

**時間**：30-45 分鐘
**參與**：編輯 + 寫手

**5 個必問**

1. 「你對藍調的認識來自哪裡？」
   - ✓ 好：具體歌曲 + 個人感受
   - ✗ 差：「都很好聽」或無例子

2. 「B.B. King vs Muddy Waters，你的第一印象？」
   - ✓ 好：對比分明（細膩 vs 粗糲）
   - ✗ 差：「都是大師」

3. 「伍佰《白鴿》聽過嗎？」
   - ✓ 好：具體音樂特徵 + 個人共鳴
   - ✗ 差：沒聽過或「很感人」

4. 「技術部分 (bending、vibrato) 熟悉度？」
   - ✓ 好：能用感受性語言描述
   - ✗ 差：「不知道」或樂理堆砌

5. 「什麼時間交初稿？」
   - ✓ 好：明確時間 (應早於 4/11 18:00)
   - ✗ 差：「大概吧」

## 初稿品質門檻 (PASS/FAIL)

### ✓ PASS 標準
- [ ] 標題 55-65 字，含主人公 + 修飾詞
- [ ] TL;DR 3-4 bullet，涵蓋核心論點、技術、社會、實用值
- [ ] 主體 6-7 個 H2，結構完整
- [ ] 字數 2,800-3,200 字
- [ ] 個人立場清晰 (2+ 處「我」「我們」)
- [ ] 台灣融合自然，佔 ~20%
- [ ] 聽覺感受描述，非樂理術語
- [ ] 無事實錯誤、無躲避表達
- [ ] 延伸閱讀 3-5 篇，連結有效

### ✗ FAIL 標準 (任一不符合 → 必改)
- [ ] 標題 > 70 字或 < 50 字
- [ ] 無主人公或無修飾詞
- [ ] 事實錯誤
- [ ] 全第三人稱，無個人聲音
- [ ] 台灣融合 < 10% 或 > 35%
- [ ] 樂理術語堆砌
- [ ] 無延伸閱讀

## 常見問題速查

### Q: 寫手交稿超期？
A: Tier 1 (Day 3 預警) → Tier 2 (Day 5 提醒) → Tier 3 (Day 6 啟動備選寫手)

### Q: 寫手技術知識不足？
A: 提供詞彙表 + 影音資料 (Level 1) → 邀請專家顧問 (Level 2) → 簡化期望 (Level 3)

### Q: 台灣融合過多/過少？
A: 計算字數比例 → 若 < 15% 加強、> 30% 削減 → 目標 20%

### Q: 發佈後排名不理想？
A: 72 小時內補救 (內容優化、社交加碼) → 7+ 天後長期監測
