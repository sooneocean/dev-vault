## I. 寫手框架分析

### 核心寫作難度評估

| 維度 | 難度 | 關鍵要求 |
|------|------|---------|
| **音樂技術知識** | ⭐⭐⭐ 中等 | 理解 12-bar blues、bending、vibrato、電吉他聲學基礎；無需深度樂理 |
| **社會歷史理解** | ⭐⭐⭐⭐ 高 | 美國種族隔離史、黑人音樂傳統、民權運動背景 |
| **台灣在地化融合** | ⭐⭐⭐ 中等 | 伍佰的藝術風格與人生故事；台灣 1990 年代搖滾背景 |
| **語調把握** | ⭐⭐⭐⭐ 高 | YOLO LAB 風格：毒舌但有據、強個人立場、短長句交錯 |
| **時間壓力** | ⭐⭐⭐ 中等 | 4 天完成 2,800-3,200 字初稿 + 修改迴圈 |

### 核心人物背景簡要

**B.B. King (1925-2015)**
- 形象：情感建築師、藍調國王、人道主義者
- 標誌特徵：vibrato 顫音大師、吉他名 Lucille、2500+ 場演出
- 代表作：《The Thrill Is Gone》(1969)、《Every Day I Have The Blues》
- 核心人生故事：民權運動見證人、從三角洲到全球舞台

**Muddy Waters (1915-1983)**
- 形象：電氣革命先驅、現代藍調之父、搖滾基因締造者
- 標誌特徵：電吉他 slide 技法、粗糲聲音、原始力量
- 代表作：《Hoochie Coochie Man》(1954)、《Mannish Boy》(1955)
- 核心人生故事：農民到藝術家、電吉他如何改變黑人音樂

**伍佰 (1968-現在) — 台灣融合點**
- 形象：台灣搖滾教父、藍調精神在地化身
- 標誌特徵：閩南語搖滾、吉他風格融合藍調、舞台魅力
- 代表作：《白鴿》(1995)、《好不容易》系列
- 融合邏輯：如何用台灣本土語言與電吉他重現 Muddy Waters 的力量

## II. 理想寫手特徵與招募方案

### 理想寫手側寫

**背景資格**（優先順序）**
1. **第一優先**：搖滾或民族音樂背景 + 能寫聽覺感受（不是樂理堆砌）
2. **第二優先**：對台灣 1990 年代音樂文化有深度認識（伍佰年代）
3. **第三優先**：理解社會歷史脈絡能力強（種族隔離史、民權運動）
4. **第四優先**：繁體中文自然流利 + 熟悉網路用語

**能力清單**
- ✓ 能聽出吉他的 bending、vibrato、slide 技法，用感受性語言描述
- ✓ 能寫出「力量感」而非只有「悲傷感」
- ✓ 對伍佰的音樂風格、人生故事有個人體驗或研究
- ✓ 能自然連結音樂與社會現象（用隱喻而非統計）
- ✓ 能寫出 YOLO LAB 風格：毒舌、個人立場、對話化語調

**背景禁區**（不考慮的類型）
- ✗ 純古典音樂或爵士背景，但對搖滾/藍調陌生
- ✗ 學術音樂評論風格（過度樂理術語）
- ✗ 企業文案風格（官方語氣、躲避表達）
- ✗ 對伍佰無認識，無法自然融合台灣案例

### Phase 3: 編輯初審 & 修改迴圈 (2026-04-11 ~ 2026-04-14)

#### 步驟 3.1: 編輯初審 (Day 6-7, 2026-04-11 ~ 2026-04-12)

**編輯 (sooneocean) 的工作**

審閱項目 (按優先級)

| 優先級 | 項目 | 檢查點 | 修改難度 |
|--------|------|--------|---------|
| **P0 (Critical)** | 內容準確性 | 歷史事實、人物生平、歌曲年份 | 高 |
| **P0** | 標題與 TL;DR | 55-65 字、包含主人公、帶修飾詞 | 高 |
| **P1** | 語調與聲音 | 是否達成 YOLO LAB 風格、有無躲避表達 | 高 |
| **P1** | 技術分析深度 | bending、vibrato、12-bar 結構是否清晰可讀 | 中 |
| **P1** | 台灣融合自然度 | 伍佰案例是否生硬、字數比例是否符合 80-20 | 高 |
| **P2** | 內部連結準備 | 延伸閱讀是否有合適的文章連結 | 中 |
| **P2** | 元數據準備 | 是否準備了 Featured Image、tags、category | 低 |

**編輯反饋的三種形式**

1. **Google Docs 註解** (即時反饋)
   - 逐段添加評論，寫手可實時看到
   - 適合小的文法、措辭問題

2. **修改建議清單** (集中反饋)
   - Gdoc 或專門文件列舉所有修改重點
   - 適合大的結構調整

3. **電話或 Zoom 討論** (複雜反饋)
   - 30-45 分鐘對話，討論核心問題
   - 適合語調、台灣融合點等複雜問題

## IV. 常見問題與解決方案

### Q1: 如何評估寫手是否適合「藍調」這個專題？

**A:** 在第一次溝通中提出以下開放式問題：
1. 「你最喜歡的藍調歌手或歌曲是什麼，為什麼？」
   - 好回答：具體歌曲 + 個人感受（表示有實際聆聽經驗）
   - 差回答：「藍調都很好聽」或無法舉例（表示認識不足）

2. 「對 B.B. King 和 Muddy Waters，你的第一印象是什麼？」
   - 好回答：對比分明（如「King 更細膩，Waters 更粗糲」）
   - 差回答：「都是藍調大師」（差異化不足）

3. 「伍佰的『白鴿』這首歌，你聽過嗎？它讓你想到什麼？」
   - 好回答：具體音樂特徵、歌詞含義、個人共鳴
   - 差回答：沒聽過或「很感人」（表示缺乏台灣音樂背景）

### Q3: 編輯反饋後，寫手拒絕修改關鍵項目怎麼辦？

- 做法：書面溝通，說明為什麼這項修改是非談判性的
- 例：「標題 70 字違反 SEO 規範，必須精簡到 65 字以內。請在 XX 時間前回傳修改版本，否則無法發佈。」

**情況 B: 寫手拒絕修改 P1 項目 (語調、台灣融合等)**

- 若編輯建議合理，但寫手堅持，可協商折中方案（e.g., 在另一位置加強個人聲音）

**情況 C: 寫手拒絕修改 P2 項目 (細部詞彙、可選項)**
- 編輯決定：P2 項目可放行，保持原文
- 不值得為小事起紛爭

### Q5: 如何處理寫手的台灣融合點過多或過少？

**計數方法**

```
1. 統計伍佰相關段落字數
   例：H2-6 中「伍佰的 [詞] 風格…」到「…台灣搖滾的DNA」 = 245 字

2. 計算整體文章字數
   例：全文 3,000 字

3. 比例 = 245 ÷ 3,000 = 8.2% (低於 20% 目標)

4. 調整：
   - 若 < 15%：加強台灣融合 (目標 + 200 字)
   - 若 15-25%：符合目標，保持
   - 若 > 30%：削減台灣融合，回到 20% (目標 - 300 字)
```

```
✓ PASS 標準：
- 標題 55-65 字，包含主人公、帶修飾詞

- 標題 > 70 字或 < 50 字

- 大部分符合但有 1-2 項需小修 → 反饋修改，預期 24 小時內完成
```
