## 使用建議

### 編輯的使用流程

```
1. 閱讀 UNIT-2-BLUES-QUICK-REFERENCE.md (5 min) — 快速上手
2. 按需查閱 UNIT-2-BLUES-WRITER-RECRUITMENT.md — 詳細執行
3. 按步驟執行 UNIT-2-BLUES-WRITER-KIT-CHECKLIST.md — 寫手包準備
4. 日常參考 Quick Reference Card — 快速檢查
```

### 寫手的使用流程

```
1. 收到寫手包 → 先讀「00_START_HERE.md」(導讀)
2. 按「優先閱讀順序」讀 4 份編輯標準 (1-2 小時)
3. 研究參考資料 (2-3 小時)
4. 聆聽音樂並邊聽邊記筆記 (3-4 小時)
5. 參考評論文章，學習寫作風格 (30-40 min)
6. 開始寫作 (4-6 小時)
7. 自審並交稿
```
