## ⏱ Phase 1：複製 & 個人化招募文案（45 分鐘）

### 工作內容
複製 5 份招募郵件範本到 Google Docs，填入每份郵件的 4 個個人化欄位。

### 快速步驟

1. **打開 Google Drive** (5 秒)
   ```
   https://drive.google.com
   ```

2. **建立新資料夾** (2 分鐘)
   - 資料夾名稱：`YOLO LAB 寫手招募 - 20世紀黑人音樂人系列 (2026-04-02)`
   - 建立子資料夾：`招募文案 & 聯繫清單`

3. **複製 5 份招募文案** (9 分鐘/份 × 5 = 45 分鐘)

   **步驟（重複 5 次）：**

   a) 在 Google Docs 建立新文件
      - 文件名：`Unit [X] 招募文案 - [樂種]`
      - 例：`Unit 1 招募文案 - Jazz`

   b) 複製對應的招募郵件範本
      - 來源：`docs/editorial/RECRUITMENT-EXECUTION-TOOLKIT.md`
      - 複製到 Google Doc 中

   c) 填入 4 個個人化欄位：
      ```
      [ ] [親愛的 [寫手名字]] → 填入: 你選定的候選人名字
      [ ] [我想到你，是因為...] → 填入: 該人選的具體背景
      [ ] [你的名字] → 填入: 你的實名
      [ ] [聯絡方式] → 填入: Email + 電話/LINE
      ```

   d) 檢查無誤，儲存

   e) 複製分享連結 (備用於 04-03 招募郵件)
