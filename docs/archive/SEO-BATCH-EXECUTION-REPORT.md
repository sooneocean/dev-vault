## 技术决策

### 为何选择 WordPress REST API 直接调用?

1. **可靠性**: 官方 API，长期支持
2. **安全性**: HTTP Basic Auth + TLS 加密
3. **可扩展性**: 支持大规模批处理
4. **可追踪性**: 每个请求都有日志
5. **原子性**: 元数据独立更新，不影响文章内容

### 为何采用队列式并发?

1. **服务器友好**: 防止 DoS 式请求
2. **稳定性**: 减少超时错误
3. **可恢复性**: 允许重试机制
4. **可控性**: 动态调整速度
