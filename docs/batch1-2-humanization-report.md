# SEO 文案人類化重寫 - Batch 1-2 分析報告

**生成時間**: 2026-04-01
**任務**: Batch 1-2 共 10 篇 SEO 優化文案的「人類化」重寫
**目標**: 移除 AI 生成痕跡，符合 YOLO LAB 真實風格

---

## AI 生成痕跡分析 (29 個模式)

### 移除的主要模式

**核心內容問題:**
- ✗ 過度重要性誇大 (significance inflation): "深度分析", "完整評測", "科學驗證"
- ✗ 模糊來源歸屬 (vague attribution): "揭秘", "分享成功達人", "發現"
- ✗ 推廣性語言冗餘: "必看", "首選", "最適合", "助您"

**語言模式問題:**
- ✗ 常用 AI 詞彙: "leverage", "explore", "utilize" → 改為直白台灣用語
- ✗ 過度 copula (be verb): "是", "了解" → 改為主動動詞
- ✗ 過度對沖 (hedging): "幫助", "可以", "一覽" → 直接陳述

**風格問題:**
- ✗ Em dash (—) 濫用: "激戰分析 —" → 改為直問
- ✗ 標題全 Title Case: "Netflix 2025 必看韓劇推薦" → 混合對話風格
- ✗ 聊天機器人用語: "I hope this helps!" (中文版: "幫你找到")

---

## YOLO LAB 風格特徵提取

### 核心特徵 (從既有高流量文章提取)

1. **語氣**: 直接、不廢話、犀利觀點，像跟朋友聊天而非企業文案
2. **在地化**: 使用台灣詞彙 (誰贏、最威、怎麼玩)、台灣梗、文化參考
3. **標題技巧**:
   - 數字錨點: "2026", "10 萬", "月入"
   - 對比詞: "vs", "誰", "怎麼選"
   - 懷疑句式: "真的假的？", "哪家最威？", "為什麼？"
4. **Meta 說話口氣**: 自然對話，不用「精選」「推薦」等模板詞
5. **去模板化**: 避免「分析」「評測」「指南」等重複詞彙

## 使用指南

### 標題改寫原則
1. **優先使用問句**：「怎麼選？」「誰最強？」「真的假的？」
2. **加入質疑詞**：「到底」「為什麼」「哪家」
3. **台灣在地詞彙**：「最威」「怎麼玩」「夠用」「划算」
4. **調整詞序**：將品牌名/工具名前置製造懸念

### Meta 改寫原則
1. **開場痛點提問**：「怕發胖？」「找工作糾結？」「不知道選哪個？」
2. **替換「幫助您」**：改用「我們」主動句式
3. **去掉企業詞彙**：「完整」「詳細」「一覽」全部去掉
4. **加入故事感**：「我們測了」「我們拆解」「實地體驗」
5. **具體而非模糊**：「月入 10 萬」vs「收入分析」

### 禁用詞清單
- ~~幫助您~~ / ~~助您~~ / ~~幫您~~
- ~~完整~~ / ~~詳細~~ / ~~全面~~
- ~~評測~~ / ~~分析~~ (改用「測」「拆解」)
- ~~推薦~~ / ~~首選~~ (改用「最推」「最威」)
- ~~精選~~ / ~~發現~~ / ~~揭秘~~
- ~~指南~~ / ~~教學~~ (改用「入門」「怎麼選」)
