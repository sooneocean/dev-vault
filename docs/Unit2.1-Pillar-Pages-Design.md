# Unit 2.1：支柱頁設計文檔

## 執行計畫

**目標**：為 3 個分類（音樂、電影、科技）各設計支柱頁，定義聚集連結拓撲。

**成果物**：
1. 3 個支柱頁 markdown 文件（可直接發佈）
2. 聚集架構圖（CSV）：支柱頁 → 文章 ID → 連結類型
3. 內部連結規劃清單（優先級排序）

## 內部連結規劃（初稿）

### 連結類型定義

| 類型 | 說明 | 數量 |
|------|------|------|
| **Pillar ← Article** | 文章連回支柱頁（「查看完整分析」） | ≤10 個 |
| **Pillar → Article** | 支柱頁連到文章（主要） | ≥40 個 |
| **Article ← → Article** | 文章間同類相關連結（次要） | ≥10 個 |

### 連結錨文本範例

**好的錨文本**（含關鍵詞）：
- 「查看 Yohee 樂評全文」
- 「Jacob Collier 編曲技巧分析」
- 「李銘順演技賞析」
- 「AI 工具對比完整指南」

**避免的錨文本**（無語義）：
- ❌ 「點這裡」
- ❌ 「了解更多」
- ❌ 「相關文章」
