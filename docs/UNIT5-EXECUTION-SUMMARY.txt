================================================================================
                          🎯 核心招募框架
================================================================================

【項目定位】
- 20 世紀黑人音樂人系列的終章（Units 1-5 匯聚點）
- 需要回顧 Jazz、Blues、Soul/R&B、Gospel 的精神遺產
- 展示嘻哈如何融合所有先辈傳統、成為當代最強社會評論工具
- 建立 YOLO LAB 在「黑人音樂綜述」領域的 SEO 權威性

【核心人物】
- James Brown（身體節奏革命） 
- Grandmaster Flash（技術與詩詞融合，《The Message》）
- Kendrick Lamar（當代社會承諾的延續者）

【台灣融合】(20% 比例)
- Yohee 《如果我是個玩咖》（台灣新勢力）
- Central Cee（跨境身份，全球華語嘻哈）
- 融合位置：H2-6（全球化與本地化）
- 字數：120-150 字

【內容結構】(8 個 H2 主題)
1. 嘻哈的孕育（320 字）— 1970s 南 Bronx 起源
2. James Brown（380 字）— 節奏革命、3000+ 采樣遺產
3. Grandmaster Flash（400 字）— DJ 技術、《The Message》
4. 聲學工藝（350 字）— 采樣、打碟、boom-bap 節奏
5. 社會診斷（420 字）— 警察暴力、毒品戰爭、身份政治
6. 全球化與本地化（380 字）— **台灣融合在此**
7. 時代對標（420 字）— **系列回顧 Units 1-4**
8. 編者點評（320 字）— 藝術地位、精神傳承、未來想像

================================================================================
                        💬 寫手要求與篩選標準
================================================================================

【必備條件】
✓ 嘻哈或當代文化背景（或 3+ 年音樂評論經驗）
✓ 既有學術深度，也有街頭真實感
✓ **能直面暴力、毒品、警察暴力等敏感議題，指出社會根源**（篩選關鍵！）
✓ 了解美國黑人社區 + 台灣城市文化
✓ 5 天內保證交稿

【面試 3 個核心問題】
1. 「過去 3 年內，你最喜歡的嘻哈藝術作品是什麼？為什麼？」
2. 「嘻哈如何融合爵士的即興、藍調的誠實、靈魂樂的深度、福音的希望？」
3. 「嘻哈談論暴力、毒品、警察暴力。你如何在文章中處理，既不迴避也不過度渲染？」

【紅旗信號】(不適合的人選)
🚩 只了解美國嘻哈，對台灣本地陌生
🚩 迴避社會敏感議題或用「藝術超越政治」簡化
🚩 無法快速進入「系列思維」
🚩 聲稱「3,500 字內無法講清楚」
