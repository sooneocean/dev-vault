# 迪士尼版權帝國的瓦解

1995 年，迪士尼買下皮克斯。2009 買下漫威。2012 買下盧卡斯影業。

30 年內，迪士尼建立了全球最強的 IP 壟斷帝國。

2026 年，這個帝國開始瓦解。

## IP 統治力的量化衰退

迪士尼掌握的 IP 在全球票房的佔比：
- 2019 年：32%
- 2023 年：28%
- 2026 年：22%（預測）

衰退速度每年 -3%。

更危險的是，迪士尼的衰退速度比全球票房衰退還快。全球衰退 -26%，迪士尼衰退 -50%。

迪士尼失去市場份額，不只是因為票房萎縮，而是因為內容吸引力在下滑。

### IP 競爭力崩潰指數

| IP | 2019 | 2026 |
|----|------|------|
| 漫威 MCU | 2.1 億 | 1.2 億 |
| 星戰 | 2.2 億 | 0.8 億 |
| 皮克斯 | 1.3 億 | 0.7 億 |

星戰首周票房跌幅 -64%。這個曾經無敵的 IP，現在連普通大片都不如。

## 為什麼漫威、星戰在衰退

### 漫威疲勞

2019 年，漫威一年產 3-4 部電影。粉絲看得過來。

2026 年，一年產 5-6 部電影 + 6-8 部劇集。總計 50+ 小時新內容。

結果：粉絲累了。品質也在下滑（前期精心打磨，現在套用模板）。

爛番茄評分從 82% 掉到 64%。

### 星戰死亡螺旋

《Rise of Skywalker》票房 12.6 億，但粉絲評分 4.3/10。

迪士尼沒有調整方向，反而推出 4 個衍生劇集試圖「修補」粉絲不滿。

結果：粉絲進一步分裂。星戰粉絲變成「原教旨主義者」，只認可盧卡斯時代。任何新劇情都被視為背叛。

迪士尼陷入死循環。

### 皮克斯的原創力喪失

皮克斯曾是動畫原創 IP 之王。現在開始做續集。《玩具總動員 3、4》《怪獸大學》《腦筋急轉彎 2》。

新原創 2020-2026 只有兩部，票房都不理想。

皮克斯變成了「動畫工廠」，不是「動畫藝術工作室」。

## 競爭對手怎樣趁虛而入

Netflix、A24、Blumhouse 在做迪士尼做不了的事：**原創 IP 建設**。

### Netflix 的原創 IP

Netflix 投入 30 億/年在原創內容。重點不是「大片」，而是「小眾 IP」。

《怪奇物語》從小眾科幻劇變成全球現象。

Netflix 可以大膽改編、承受粉絲反彈。失敗了，下個季度做新 IP。

### A24 的獨立帝國

A24 放棄大片，做「藝術電影+怪誕電影」。

在年輕觀眾中的文化滲透率已經超過迪士尼。

## 迪士尼的出路

### 路 1：激進重組（風險極高）

拆分成「IP 部門」和「製作部門」。IP 部門授權，製作部門創新。

代價：內部戰爭、股東反彈、短期股價暴跌。

### 路 2：漸進改良（保守但困難）

逐步減少續集，增加原創投資。需要 5-10 年，華爾街等不了。

### 路 3：放棄（最可能）

迪士尼持續虧錢，最終被收購或分拆。IP 變成古董。
