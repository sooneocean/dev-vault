# YOLOLAB.net SEO 批量优化计划

## 任务概述
- 站点：yololab.net (Site ID: 133512998)
- 总文章数：2,725 篇
- 已优化：74 篇
- 待优化：2,651 篇

## 执行策略

### 第一阶段：样本验证（3-5 篇文章）
1. 获取前 5 篇文章的完整数据
2. 生成优化的 SEO 标题（45-60 字）
3. 生成优化的 Meta 描述（120-160 字）
4. 通过 WordPress MCP 的 posts.update 操作更新元数据
5. 验证更新是否成功

### 第二阶段：批量处理（每批 10 篇）
- 分页获取文章列表
- 为每篇文章调用 Claude API 生成 SEO 元数据
- 使用 posts.update 操作更新数据
- 每 100 篇后报告进度

### 第三阶段：完整运行
- 处理所有 2,651 篇待优化文章
- 追踪失败案例
- 生成最终统计报告

## 技术实现

### 工具链
1. **WordPress MCP**: posts.list, posts.get, posts.update
2. **Anthropic API**: Claude Opus 4.6 用于 SEO 生成
3. **Node.js**: 脚本编排和 API 调用

### SEO 优化规范

#### SEO 标题
- 长度：45-60 字
- 包含主要关键词
- 第一人称或命令式语气
- 包含数字、年份、地点等具体信息

#### Meta 描述
- 长度：120-160 字
- 包含 2-3 个关键词
- 包含呼籍行动 (CTA)
- 清晰、有吸引力的表述
