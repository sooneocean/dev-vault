## 執行風險與應對

### 風險 1：排名下降
**可能原因：** Google 演算法更新或競品加強
**應對方案：**
- 強化內部連結（指向權重高的頁面）
- 增加外部連結或提及
- 優化頁面速度和可讀性

### 風險 2：CTR 低於預期
**可能原因：** Meta description 吸引力不足、或排名位置太後
**應對方案：**
- A/B 測試不同 description 風格
- 優化標題引導力
- 增加內部連結以提升排名

### 風險 3：新文章不收錄
**可能原因：** 伺服器問題、或 robots.txt 阻擋
**應對方案：**
- 在 GSC 中手動提交 URL
- 檢查 robots.txt 和 sitemap
- 檢查頁面沒有 noindex 標籤
