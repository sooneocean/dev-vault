## ⚠️ 已知限制

### 1. 外掛管理 API 限制

**限制**：WordPress.com 不支持通過 REST API 停用/啟用外掛

**影響**：SpeedyCache 和 Page Optimize 需手動停用

**解決**：詳見 `docs/MANUAL-STEPS.md` 第 1 步

**狀態**：已記錄為已知限制，文檔已提供

### 2. CDN 快取延遲

**限制**：全球 CDN 節點同步需 48-72 小時

**影響**：性能提升需時間觀察

**解決**：在指定時間後重新測試

**狀態**：正常行為，非問題

### 3. Gzip 頭檢測延遲

**限制**：初始測試可能未顯示 Content-Encoding 頭（CDN 快取）

**影響**：Gzip 已啟用但頭可能暫時不顯示

**解決**：24 小時後重新檢查或直接測試性能

**狀態**：預期行為
